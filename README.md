# Peak Mind Mindfulness App

A cross-platform mobile app (iOS/Android) built with React Native and Expo that guides users through Dr. Amishi Jha's six-week mindfulness training program.

## Overview

This app implements the Peak Mind practice guide, featuring four core practices:
- **Find Your Flashlight** - Breath awareness meditation
- **Body Scan** - Systematic body awareness practice
- **River of Thought** - Meta-awareness meditation
- **Connection Practice** - Loving-kindness meditation
