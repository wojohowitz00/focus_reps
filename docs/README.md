# Mindfulness App

A cross-platform mobile app (iOS/Android) built with React Native and Expo that guides users through a research-based six-week mindfulness training program.

## Overview

This app implements evidence-based mindfulness practices derived from peer-reviewed research, featuring four core practices:
- **Breath Focus** - Focused Attention (FA) meditation
- **Body Awareness** - Systematic body awareness practice
- **Open Awareness** - Open Monitoring (OM) meditation
- **Compassion** - Loving-Kindness Meditation (LKM)
